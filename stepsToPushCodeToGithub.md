## STEPS TO PUSH CODE TO GITHUB
 - Create Folder for code
 `mkdir nameOfFolder`
 - Create local Repo
 `git init`
 - Create code
 `touch readme.md`
 - Track/State/Watch all the files/folder
 ```
    git add .
```
or 
```
    git add -A
```
- Commit our changes to our local git
```
    git commit -m "first commit"
```
